# 🔱 Hoja de Ruta: Grit App "Ultimate Edition"

## 1. Gamificación y Recompensas
- [x] **Sistema de Logros (Badges)**: Medallas visuales por hitos.
- [x] **Tienda de Skins Real**: Desbloqueo de temas mediante XP.
- [x] **Efectos de Nivel**: Modal cinemático `LevelUpModal`.
- [x] **Misiones Diarias**: Retos aleatorios para ganar XP.
- [x] **Batalla contra el Boss**: Evento semanal donde tus hábitos atacan a un jefe. (NUEVO)

## 2. Sistema Social y Competencia
- [x] **Leaderboard Global**: Ranking de Guerreros Grit por nivel y XP (Supabase Sync).
- [x] **Feed de la Tribu**: Espacio social para publicar reportes de guerra y dar Hype 🔥. (NUEVO)
- [ ] **Duelos de Disciplina**: Retar a amigos a cumplir hábitos.
- [ ] **Feed de Actividad**: Ver el progreso detallado de amigos.

## 3. Notificaciones e Inteligencia
- [x] **Recordatorios Inteligentes**: Notificaciones push con frases motivadoras.
- [x] **Notificaciones de "Racha en Peligro"**: Aviso si no has cumplido misiones (lógica en `checkStreakStatus`).
- [ ] **Resumen Semanal**: Reporte dominical con % de éxito.

## 4. Funciones de Seguimiento Avanzado
- [x] **Módulo de Abstinencia Real**: Contador de tiempo real para vicios con botón de "Recaída".
- [x] **Rutinas y Programación**: Bloques de tiempo (Mañana, Tarde, Noche) y días específicos. (NUEVO)
- [x] **Notas en los Hábitos**: Diario de reflexión y notas motivadoras por cada hábito.
- [ ] **Widget de Pantalla de Inicio**: Ver racha y hábitos en el escritorio.
