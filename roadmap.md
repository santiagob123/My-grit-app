# 🔱 Hoja de Ruta: Grit App "Ultimate Edition"

## 1. Gamificación y Recompensas
- [x] **Sistema de Logros (Badges)**: Medallas visuales por hitos (🌱 Iniciado, 🔥 Constante, etc.). Implementado con galería y alertas.
- [x] **Tienda de Skins Real**: Desbloqueo de temas (Minecraft, Fútbol) mediante XP y niveles.
- [ ] **Efectos de Nivel**: Animaciones especiales al subir de rango. (PRÓXIMO PASO)
- [x] **Misiones Diarias**: Retos aleatorios para ganar XP extra. (RECIÉN AÑADIDO)

## 2. Sistema Social y Competencia
- [ ] **Leaderboard Global**: Ranking mundial de Guerreros Grit.
- [ ] **Duelos de Disciplina**: Retar a amigos a cumplir hábitos.
- [ ] **Feed de Actividad**: Ver el progreso de amigos.

## 3. Notificaciones e Inteligencia
- [x] **Recordatorios Inteligentes**: Notificaciones push con frases motivadoras agresivas.
- [ ] **Notificaciones de "Racha en Peligro"**: Aviso si no has cumplido misiones a las 8:00 PM.
- [ ] **Resumen Semanal**: Reporte dominical con % de éxito.

## 4. Funciones de Seguimiento Avanzado
- [x] **Módulo de Abstinencia Real**: Contador de tiempo real para vicios con botón de "Recaída" y Multimisión.
- [ ] **Notas en los Hábitos**: Diario de reflexión por cada hábito.
- [ ] **Widget de Pantalla de Inicio**: Ver racha y hábitos en el escritorio.

## 5. Contenido y Experiencia de Usuario (UX)
- [ ] **Audios Motivacionales**: Clips de audio al completar tareas.
- [x] **Gráficas de Análisis**: Visualización de progreso en Stats. (RECIÉN AÑADIDO)
